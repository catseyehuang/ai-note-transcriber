# AI 筆記助理 (AI Notes Assistant) 📝✨

這是一個專門為會議與學習者設計的**單頁 Web 應用程式 (SPA)**。它能幫助您在開會或上課時進行現場錄音，並結合系統內建的語音輸入功能（Windows 的 `Win + H` 與 Mac 的系統聽寫）即時生成逐字稿，最後透過 **Google Gemini API** 一鍵將粗糙的文字整理成結構化、排版美觀且適合直接貼入 **Notion** 的 Markdown 智慧筆記。

---

## 🌟 核心特色

*   **📂 歷史會話紀錄 (Session History)**：支援建立多個獨立會話。您可點擊「新增會話」開啟全新記錄，並在側邊欄切換或刪除歷史紀錄。
*   **✏️ 連按兩下重命名會話**：對側邊欄的會話項目連按兩下，即可自訂該會話的名稱，方便日後管理與檢視。
*   **🎙️ 現場音訊錄製**：使用瀏覽器 Web Audio API，無需安裝額外軟體。支援暫停、繼續、結束錄製，並提供音訊播放與 WebM 格式下載。
*   **📊 動態音波視覺化**：未錄音時呈現優美的正弦波待命畫面；錄音時會即時分析音訊頻率，繪製出紫/青霓虹漸層的 3D 動態對稱頻率圖。
*   **💻 雙系統語音輸入支援**：詳細說明並引導使用者點擊輸入框後啟動語音輸入：
    *   **Windows 用戶**：按 <kbd>Win</kbd> + <kbd>H</kbd> 鍵。
    *   **Mac 用戶**：連按兩下 <kbd>Control</kbd> (⌃) 鍵或按一次 <kbd>🌐</kbd> (Fn) 鍵。
*   **💾 逐字稿與筆記下載**：
    *   一鍵下載原始逐字稿為 `.txt` 文字檔。
    *   一鍵下載 AI 筆記為 `.md` Markdown 檔案。
*   **🤖 一鍵 Gemini AI 整理**：直連 Google 官方 `gemini-2.5-flash` 模型，支援多種筆記風格（系統化整理筆記、**會議記錄與決議事項**、精簡摘要、心智圖大綱、重點 Q&A），自動修正錯字與贅詞。
*   **⚡ LocalStorage 自動存檔**：逐字稿、會話紀錄、設定等資訊皆即時保存至本機瀏覽器快取，防斷電、網頁重新整理或誤關。
*   **🔒 100% 隱私與安全**：全前端架構，您的 Gemini API Key 只儲存在您的本機瀏覽器，直接與 Google API 通訊，絕不經過任何第三方伺服器。
*   **📋 Notion 友善匯出**：一鍵複製的 Markdown 與 Notion 的 Block 語法高度相容。直接在 Notion 頁面中 `Ctrl + V` 貼上，標題、表格、待辦清單、引言會自動轉換成 Notion 元件！
*   **🔌 零依賴本機伺服器**：隨附以 PowerShell 寫成的極簡 HTTP 伺服器，按兩下即可在 localhost 安全上下文中執行（啟用麥克風權限）。

## 🚀 快速開始

### 方式一：直接執行（雙擊啟動，推薦）
1. 下載本專案所有檔案至您的 Windows 電腦。
2. 對 **`start.bat`** 檔案按滑鼠左鍵**兩下**執行。
3. 系統會自動啟動本機伺服器並在您的瀏覽器打開 `http://localhost:8000/`。

### 方式二：手動在終端機啟動
若您想手動透過 PowerShell 啟動：
```powershell
# 切換至專案目錄並執行
powershell -ExecutionPolicy Bypass -File .\serve.ps1
```

### 方式三：Mac 用戶啟動
Mac 已預設安裝 Python，您可直接啟動本機伺服器：
1. 打開 Mac 終端機 (Terminal)。
2. `cd` 切換至專案目錄。
3. 執行指令：`python3 -m http.server 8000`。
4. 打開瀏覽器存取：`http://localhost:8000/`。

## 📝 筆記與會議整理流程

1.  **開啟網頁**並點選右上角 **「Gemini API 設定」** 輸入您的 Gemini Key（以 `AIzaSy` 開頭）。
2.  點選 **「開始錄音」**（作為音訊備份）並允許麥克風權限。
3.  點擊 **「即時逐字稿聽寫」輸入框** 使游標閃爍。
4.  按下語音輸入快捷鍵（Win+H 或 Mac 雙擊 Control）開始語音聽寫，自動在框中生成文字。
5.  結束後，點擊 **「結束錄音」**，選擇想要的 **「筆記風格」**（如：會議記錄與決議事項）。
6.  點擊 **「透過 AI 整理筆記」**，等待 10-15 秒生成完成。
7.  點擊 **「複製至 Notion」**，並貼到您的 Notion 頁面中即完成！
8.  *雙擊側邊欄的會話卡片即可重命名該次會議/學習紀錄。*

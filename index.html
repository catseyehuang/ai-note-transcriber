快速上手指南

  1. 錄製音訊（可選）：

    在左側的錄音控制區，點選「開始錄音」以錄製課堂現場聲音。這可以作為您課後重聽與溫習的音訊備份。

  2. 啟動 Windows 語音輸入（核心）：

    點擊左下角的「課堂逐字稿輸入框」，使其處於輸入狀態。接著按下鍵盤的 Windows 鍵 + H，啟動 Windows 系統內建的語音聽寫功能。開始上課後，電腦就會自動將聽到的聲音轉化為文字打入框中。

  3. AI 整理與預覽：

    錄音結束後，選擇您想要的「筆記風格」，點擊「透過 AI 整理筆記」。AI 會自動將粗糙的語音文字重新梳理，產生排版優美的 Markdown 筆記。

  4. 匯出至 Notion：

    點選右下角的「複製筆記」，然後在您的 Notion 頁面中直接按下 Ctrl + V 貼上，Notion 會自動將 Markdown 解析為漂亮的區塊樣式！
